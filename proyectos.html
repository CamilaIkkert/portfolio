  • Farmacia Raspo

    Este proyecto fue realizado como trabajo final para un curso de desarrollo Web. El mismo fue creado a partir de HTML, CSS Y Bootstrap. También se utilizó Sass de manera tal que mejore el procesamiento de CSS y Git para el control de las versiones del código. La página Web cuenta con 5 secciones, una barra de navegación, dos formularios de pedidos, iframes y tablas.

  • Caza al ratón!

    Juego creado para un curso de Javascript. Fue creado a partir de HTML, CSS y JS. También se utilizó la librería JQuery.
    El juego es una single page en la que se puede seleccionar la dificultad del mismo para luego comenzar el mismo. Además posee contador de vidas, el cual una vez terminado, finaliza el juego mostrando los puntajes, el nivel alcanzado, puntería, etc.

  • Ecommerce TikiMugs
